# Build env requirements
- tailwindcss binary 
- mupdf needs:
    - Alpine Linux: fontconfig-dev
    - Arch Linux: fontconfig
    - Debian-based systems: libfontconfig1-dev
    - FreeBSD: fontconfig
    - Nix: fontconfig, pkg-config (tested on nixos-unstable)
    - Void Linux: fontconfig-devel
